# Alpha Keystorm

**Live Demo:** [Play Alpha Keystorm Here](https://alphakeystorm.netlify.app/)

Welcome to Alpha Keystorm, a fast-paced, browser-based typing game designed to test your speed and accuracy. The core concept and foundational logic of this project were independently developed and 100% directed by me.

## The Development Story: Core Engineering & AI Assistance

This project is a result of my original logic building combined with strategic AI utilization. The primary idea, structure, and direction are entirely my own.

* **Logic & Concept:** The core physics and mechanics were conceptualized and 100% directed by me.
* **AI Agents Used:** I utilized **Google Anti-Gravity** as the primary pure AI agent to write the core code based strictly on my instructions. Additionally, I used **Google Gemini** to generate prompts, structure the `index.html` file, and assist with debugging.
* **Intensive Debugging:** Developing this wasn't a one-click process. It involved over 2 to 3 hours of rigorous manual review. I personally identified edge cases, directed the prompts, and manually fixed approximately 95% of the core errors to make the game functional.
* **Performance & Optimization:**
    * **Desktop:** The application is highly optimized and runs flawlessly on desktop browsers, delivering a smooth and fast experience.
    * **Mobile:** Please note that the game is not yet 100% optimized for mobile devices. Initial versions were heavy and caused system heating due to complex DOM manipulations. I have manually optimized the code to make it significantly more lightweight and prevent thermal issues, but full mobile optimization is still a work in progress.

## Features

* **Real-time Keystroke Tracking:** Accurately registers user input against dynamic, moving targets.
* **Anti-Gravity Physics:** Custom logic to simulate floating and falling elements on the screen.
* **Zero Dependencies:** Built entirely with raw HTML, CSS, and Vanilla JavaScript (contained within a single index.html file) for quick execution.

## Tech Stack

* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Development Approach:** 100% Manual Direction + Google Anti-Gravity (Code Generation) + Gemini (Prompting/Debugging)
* **Deployment:** Netlify

## How to Run Locally

Since the application runs entirely on client-side logic, execution is simple:

1. Clone this repository to your local machine.
2. Navigate to the game directory.
3. Open the `index.html` file directly in your preferred web browser.
